# Memory Review Checklist

- [ ] This note contains no secrets, credentials, customer data, or personal data.
- [ ] The observation is supported by evidence.
- [ ] The note is not merely a temporary workaround.
- [ ] The proposed destination is correct.
- [ ] Existing docs were checked for duplication or contradiction.
- [ ] The final wording is actionable for future agents.
- [ ] The natural-language content is mostly Japanese; English is limited to identifiers, commands, product names, logs, quotes, and short technical terms.
- [ ] The note has been marked as promoted, archived, or rejected.
